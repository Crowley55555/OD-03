# Алгоритмы сортировки на Python
Этот учебный проект демонстрирует реализацию 4 популярных алгоритмов сортировки на Python:

## Реализованные алгоритмы
###Пузырьковая сортировка (Bubble Sort)

- Простой алгоритм с временной сложностью O(n²)

- Многократно сравнивает и меняет местами соседние элементы

### Быстрая сортировка (Quick Sort)

- Эффективный алгоритм с временной сложностью O(n log n) в среднем случае

- Использует стратегию "разделяй и властвуй" с опорным элементом

### Сортировка выбором (Selection Sort)

- Алгоритм с временной сложностью O(n²)

- На каждом проходе находит минимальный элемент и помещает его в начало

### Сортировка вставками (Insertion Sort)

- Алгоритм с временной сложностью O(n²)

- Постепенно строит отсортированную часть массива, вставляя элементы на нужные места

## Использование
Каждый алгоритм реализован в виде отдельного блока кода с примером использования. Для тестирования можно заменить примеры данных в вызовах функций.
